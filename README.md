###  $ IMC = Peso / altura^2 $

| IMC | Classificação |
| ------------- | ------------- |
| < 16 |  Magreza Grave |
| 16 a < 17 | Magreza  Moderada|
| 17 a < 18.5 | Magreza Leve |
| 18.5 a 25   | Saudável |
| 25 a < 30  | Sobrepeso |
| 30 a < 35  | Sobrepeso Grau 1 |
| 35 a < 40  | Obesidade Grau 2 **(Severa)** |
|  >= 40 | Obesidade Grau 3 **(Mórbida)**  |
